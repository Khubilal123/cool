# Campus Lost and Found Portal

## Overview
The Campus Lost and Found Portal is a web application designed to help students and staff report and find lost items on campus. The portal provides a user-friendly interface for submitting lost and found items, viewing available items, and managing user interactions.

## Features
- Submit lost items with descriptions and images.
- Browse found items posted by other users.
- Search functionality to quickly find specific items.
- Responsive design for mobile and desktop users.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd campus-lost-found-portal
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the local dev server and open the app in your browser:
   ```
   npm start
   ```

The app's main page is `index.html` at the project root. Styles are in `src/css/style.css` and client JS is `main.js`.

## Usage Guidelines
- To submit a lost item, fill out the form on the homepage and provide as much detail as possible.
- To view found items, navigate to the "Found Items" section.
- Use the search bar to quickly locate specific items.
